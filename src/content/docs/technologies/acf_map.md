## Description

ACF Map est une fonctionnalité du plugin Advanced Custom Field (ACF) qui permet d'ajouter un champ Google Maps dans j'administration WordPress.<br>
Ce champ permet d'enregistrer des informations géographiques (latitude, longitude, adresse) et d'afficher ensuite une carte perconnalisée sur le site.

Il est principalement utilisé pour

* Pages de contact
* Localisation d'entreprise
* Agences multiples
* Biens immobiliers
* Evénements

### Possibilités du champ ACF Map

##### Sélection de localisation

* Recherche d'adresse via Google Maps
* Placement manuel d'un marqueur
* Enregistrement automatique des coordonnées GPS
* Stockage de l'adresse complète

##### Affichage dynamique sur le site

* Affichage d'une carte avec marqueir personnalisé
* Zoom automatique selon la position
* Possibilité d'afficher plusieurs marqueurs
* Intégration dans des templates personnalisés

##### Intégration avec le thème WordPress

Le champ Map peut être affiché

* Dans un template PHP personnalisé
* Dans une page spécifique
* Dans une boucle pour afficher plusieurs emplacements
* Avec du JavaScript personnalisé

## Customisation réalisée

Dans mon projet, j'ai efectué une personnalisation avancée de la carte

* Modification du style visuel de la carte
* Personnalisation du marqueur
* Réglage du niveau de zoom
* Ajout d'information au clic sur le marqueur
* Adaptation responsive pour mobile

## Avantages de ACF Map

* Gestion simple depuis l'administration
* Données stockées proprement dans WordPress
* Flexivle et adaptable
* Compatible avec les thème personnalisés
* Idéal pour les sites professionnels

## Conclusion

Le champ Map de Advanced Custom Field permet d'intégrer facilement une carte interactive et dynamique dans un site WordPress. Grâce à la personnalisation réalisée, la carte ne sert pas uniquement à afficher une localisation, mais devient un véritable élément intégré au design et à l'expérience utilisateur du site.
