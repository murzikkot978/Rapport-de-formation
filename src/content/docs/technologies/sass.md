## Description

CSS (Cascading Style Sheets) permet de définir le style visuel d’un site web (couleurs, polices, marges, etc.).
SASS (Syntactically Awesome Style Sheets) est un préprocesseur CSS qui facilite l’écriture du code grâce à des variables, mixins et fichiers partiels.

## Utilisation pendant la formation

J’ai utilisé le **CSS** et le **SASS** pour personnaliser le design du thème WordPress.
J’ai créé des fichiers partiels pour organiser le style du site (typographie, couleurs, mise en page) et utilisé des variables pour garantir la cohérence graphique.

* Structuration d'un projet CSS avec SASS.
* Utilisation de variables et mixins pour un cod plus propre et réutilisable.
