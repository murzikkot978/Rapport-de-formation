Ce projet permet à l'utilisateur d'enregistrer ses tâches sur une page web.

L'utilisateur peut ajouter une date limite, modifier l'état lorsque le bouton est cliqué, ajouter un type de tâche.

Au début, j'ai utilisé **_localstorage_** pour sauvegarder les données.

Ensuite, j'ai changé l'enregistrement pour le **_serveur local_**.

L'enregistrement sur le serveur est effectué à l'aide de **_Fetch_**, mais avant cela, le type de données est remplacé par le type **_JSON_**.
