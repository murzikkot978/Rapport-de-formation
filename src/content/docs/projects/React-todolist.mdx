### Description
Il s’agit d’une application web de liste de tâches simple et interactive qui permet aux utilisateurs de gérer efficacement leurs tâches.
Les utilisateurs peuvent ajouter, supprimer et trier les tâches selon différents critères.
L’application propose également une gestion des catégories avec des couleurs personnalisables pour chaque catégorie et stocke les données via une API.

### Fonctionnalités
**Ajouter une tâche :** Les utilisateurs peuvent ajouter des tâches avec un titre, une date d’échéance et une catégorie.

**Supprimer une tâche :** Les utilisateurs peuvent supprimer une tâche avec le bouton (supprimer) ou supprimer toutes les tâches avec le bouton (tout supprimer).

**Changer le statut :** Les utilisateurs peuvent changer le statut de chaque tâche à l’aide d’une case à cocher.

**Modifier le titre et la date de chaque tâche :** Les utilisateurs peuvent modifier le titre et la date d’échéance en cliquant sur la tâche.

**Trier les tâches :** Les tâches peuvent être triées par nom, date ou statut (fait/non fait).

**Gestion des catégories :** L’utilisateur peut sélectionner une catégorie pour chaque tâche.
